Proyek : Pendataan Nilai Ujian Mahasiswa

Anggota : Muhammad Alif Rafi (003), Bellanice Bonifacia Siahaan (030), Adrian Ricardo Sipahutar 

Deskripsi Singkat : Program pendataan nilai ujian mahasiswa ini dibuat untuk membantu proses pengelolaan data nilai agar lebih mudah, cepat, dan terstruktur. Program ini memungkinkan pengguna untuk memasukkan data berupa nama, NIM, dan nilai mahasiswa, kemudian menampilkannya kembali dalam bentuk daftar. Program juga menyediakan fitur pencarian data mahasiswa berdasarkan NIM, perhitungan rata-rata nilai, serta pengurutan ranking nilai dari yang tertinggi ke terendah. Selain itu, tersedia fungsi untuk menampilkan tiga mahasiswa dengan nilai tertinggi. Dengan adanya program ini, proses pendataan nilai yang biasanya dilakukan secara manual dapat dilakukan secara lebih efisien dan sistematis.
